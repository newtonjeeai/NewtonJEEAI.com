// Week by Week
Full Curriculum
  • Setting up Python, Jupyter Notebook, VS Code & Git
  • Data types: strings, lists, tuples, dicts, sets
  • Control flow: if/else, for/while loops
  • Functions, lambda, map/filter/zip
  • File I/O: reading CSVs, JSON, text files
  • OOP basics: classes, objects, inheritance
  • NumPy arrays, shapes, broadcasting, vectorised ops
  • Creating & importing DataFrames
  • Indexing, slicing, boolean masking
  • Handling missing data — fillna, dropna, imputation
  • GroupBy, pivot tables, merging & joining
  • String operations & datetime handling
  • Matplotlib: line, bar, scatter, histogram, heatmap
  • Seaborn: pairplot, boxplot, violinplot, correlation
  • Plotly interactive charts
  • EDA workflow on a real-world Kaggle dataset
  • Outlier detection & distribution analysis
  • Communicating insights with charts
  • ML pipeline: train/test split, feature scaling
  • Linear & Logistic Regression with Scikit-learn
  • Model evaluation: accuracy, precision, recall, F1
  • Cross-validation basics
  • Capstone: End-to-end prediction project
  • Code review, portfolio submission, badge issuance
